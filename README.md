# 🚀 TaskFlow

A modern full-stack Task Management application built using **FastAPI** and **React**.

> Currently under active development.

---

## 📌 Overview

TaskFlow helps users organize their daily tasks with a clean, responsive interface and a powerful backend API.

This project is being developed to learn professional full-stack development while following industry best practices.

---

- Pydantic
